# Reserved Control Names

For more information on Control Naming standards, see the [Code Style Guide](Style_Guide.htm).

#### Status

A control to reflect the connection status of a controlled device.

#### IPAddress

Optional. A control that allows users to input an IP address for connecting to the device.

#### MACAddress

Optional. A control to display the MAC Address of the controlled device when supported by the device's API

#### Username

Optional. A control that allows users to input a username for accessing the API or opening a socket when required by the controlled device.

#### Password

Optional. A control that allows users to input a password for accessing the API or opening a socket when required by the controlled device.

#### DeviceName

Optional. A control that displays (or when applicable, sets) the name of the controlled device when supported by the device's API.

#### SerialNumber

Optional. A control that displays the serial number of the controlled device when supported by the device's API.

#### DeviceFirmware

Optional. A control that displays the current firmware version of the controlled device when supported by the device's API.

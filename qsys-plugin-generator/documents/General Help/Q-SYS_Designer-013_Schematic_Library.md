# Schematic Library

The Q-SYS **Schematic Library** contains virtual DSP Components you can use to design and test your audio system. The Schematic Library is located in the right-side pane below Graphic Tools, and the Properties of any selected component.

The Components in the library are selected and configured during the design phase of the system and are used for adjusting and controlling various aspects of the audio during the run mode. Components are placed in the **Schematic** by selecting and dragging the Component into the **Schematic**. They are then connected, configured and set up as needed.

# Q-SYS Designer Software Overview

Q-SYS Designer Software (QDS) is the design software application you use to create the design file which is loaded onto a Q-SYS Core processor.

This section covers the following topics:

#### [Installing and Updating Q-SYS Software and Firmware](0015_Installing_Q-SYS.htm)

The Q-SYS Designer software is installed on a PC, and is used for creating and maintaining the design file for your audio system. The Q-SYS Firmware is installed on Q-SYS specific hardware.

#### [Getting Started](002_Getting_Started.htm)

This topic describes the Q-SYS Designer workspace and basic functionality of Q-SYS Designer.

#### [Emulate Mode](003_Emulate_Mode.htm)

In the Emulate mode you can set control values, write script, and test control logic. You cannot pass audio. You do not have to be connected to the network.

#### [Loading and Saving Designs](../Schematic_Library/load_from_core.htm)

This topic describes how to load a running design from a connected Core and save a design to a Core.

#### [Using the Schematic](Using_the_Schematic/Using_the_Schematic.htm)

The Schematic is where the design is created. This topic describes the functionality of the schematic, including tips and tricks you can use in the schematic.

* [About Schematic Elements](Using_the_Schematic/About_Schematic_Elements.htm) - Schematic Elements are any items in the schematic. This section covers general characteristics of items in the schematic.
* [Signal Pins and Wiring in the Schematic](Using_the_Schematic/Connections_and_Wiring_in_the_Schematic.htm) - Once an item is placed in the Schematic, it needs to be connected to other Schematic Elements to complete your design. There are various rules and ways to make those connections. This topic covers the rules for making connections, and the various ways to make the connections.
* [Labeling and Organizing Schematic Elements](Using_the_Schematic/Labeling_and_Organizing_Schematic_Elements.htm) - Designs can be very large, covering multiple Schematic Pages. This section describes various ways provided by Q-SYS Designer to identify and organize your design.
* [Graphic Tools](014_Graphic_Tools.htm) - Q-SYS Designer provide several tools to create graphical shapes in your design. This topic covers the details of creating the graphics, and how they can be used.

#### [Inventory](../Schematic_Library/inventory.htm)

The Q-SYS Designer Inventory section contains virtual components representing physical hardware. This topic covers how to add hardware, and how use the Inventory.

#### [Schematic Pages](../Schematic_Library/schematic_pages.htm)

Q-SYS Designer provides the capability of creating multiple pages of schematic for your design. This topic describes how to create and use Schematic Pages.

#### [User Control Interfaces](../Schematic_Library/user_control_interface.htm)

This topic provides the procedure for creating and deploying user interfaces.

#### [Snapshots](../Schematic_Library/snapshots.htm)

This topic describes how to save the settings of controls within your design and recall them at a later time.

#### [Named Controls](../Schematic_Library/external_control.htm)

This topic describes how you can create a list of controls that are available to external control systems, [Control Commands](../Administrator/Commands.htm), and [Control Scripting](../Schematic_Library/control_script_2.htm).

#### [Schematic Library](013_Schematic_Library.htm)

The Schematic Library contains all the virtual components used to create designs. This topic lists the components available in Q-SYS Designer, and provides links to the associated topics.

#### [User Library](user_library.htm)

Allows you to drag frequently used, pre-configured components into the User Library and have them available for use in any design you open.

#### [Check Design](016_Check_Design.htm)

This topic describes how you can quickly check various usage aspects of your design, including DSP, network audio input and output, and network bandwidth.

#### [Remote Access with Q-SYS Designer](../Networking/Remote_Connectivity.htm)

This topic describes how you can configure your PC to allow Q-SYS Designer to discover and connect to a remotely located Core.

# Q-SYS Compatibility

Releases of Q-SYS Designer Software (QDS) sometimes add support for new hardware/software and remove support for End of Life (EOL) hardware/software, including Q-SYS Core processors, peripherals, loudspeakers, accessories, and QDS software versions. As a rule:

* Hardware introduced in a release is compatible with future QDS releases until the hardware is deprecated and no longer supported.
* Hardware introduced in a release is not backward-compatible with previous releases.
* Software for [Discontinued Products](https://www.qsys.com/support/discontinued-products/) will be supported throughout the standard warranty period. However, features introduced in newer QDS releases may not be compatible with discontinued products.

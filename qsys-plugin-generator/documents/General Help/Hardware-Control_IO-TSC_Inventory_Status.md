# TSC Inventory Status Screen

This section details the functions available on the TSC hardware Inventory Status page. This page is used to access hardware information and run utilities / diagnostics as needed on the device.

The Inventory Status page can be accessed in two ways:

* Swiping up from the bottom of the TSC panel
* Selecting the Inventory Status from the UCI Control section of the TSC Status component

### Status Information

#### Name

Displays the current name of the TSC unit.

#### LAN A

Displays the current IP address of the device.

#### Design

Displays the current running Q-SYS design.

#### UCI

Displays the current UCI

### Utility Controls

#### Brightness Control Slider

This slide control is used to adjust the screen brightness.

#### ID

Locates the TSC in Q-SYS Designer and Configurator.

#### Clean Screen

This button starts a 30 second timer allowing you to clean the screen.

#### Log Off

When UCI Access is configured (see [User Control Interfaces](../../Core_Manager/System_Management/User_Control_Interfaces.htm#Configur)), this will log out the current user.

#### Screen Test

Runs diagnostic tests to verify screen functionality.

#### Factory Reset

Use to reset the TSC back to its factory default settings. You will be prompted to confirm reset before process proceeds.

#### Close

Closes the Inventory Status screen

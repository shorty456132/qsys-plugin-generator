# RoomSuite Modular System Release Notes

## v1.0.2

v1.0.2 is the initial release of the Q-SYS RoomSuite Modular System, enabling fast, standardized setup for medium to large collaboration spaces. Designed to work with leading UC platforms and BYOM functionality, it offers a web-based workflow for efficient design, commissioning, and deployment, supported by unified cloud management and remote monitoring through Q-SYS Reflect.

For more information, please visit [Q-SYS RoomSuite Modular System](RoomSuite_Overview.htm) .

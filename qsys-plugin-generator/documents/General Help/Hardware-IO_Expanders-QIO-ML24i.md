# QIO-ML24i

The Q-SYS QIO-ML24i is a network audio endpoint native to the Q-SYS Ecosystem, serving as a set of mic/line inputs that enables network-based audio distribution. The form factor of this product facilitates a high density of 24 analog audio connections in a standard 1U rack space, ideal for applications that require a large number of audio connections. The QIO-ML24i offers 24 mic/line input connections along with 8x8 GPIO ports for control. Dual Ethernet connectors provide optional network redundancy, and two of these products can also be installed in a redundant failover pair to guard against unexpected audio interruptions in critical applications.

**Note:** This topic provides an overview of the QIO-ML24i. For specifications and installation documentation, see the [QIO Series High-density Audio Expanders product page](https://www.qsys.com/products/q-sys/audio-io-peripherals/hdqio/) on the Q-SYS website.

[Features](javascript:void(0))

* Native, high-density audio I/O for Q-SYS
* Twenty-four (24) mic/line level inputs
* Supports both network and peripheral redundancy
* 1RU form factor
* 8 x 8 GPIO to integrate third-party device control connectivity
* Configurable +48 VDC phantom power on inputs
* Microphone detection on inputs enables monitoring, usage statistics, and failure notifications
* Two (2) gigabit Ethernet ports for redundant networked audio
* Surface- or rack-mountable
* QIO-CK redundancy cable kit (sold separately)

[Front Panel - QIO-24f, QIO-ML24i, QIO-L24o](javascript:void(0))

1. Air intake vents
2. ID button invokes ID feature
3. ID LED illuminates when ID feature is invoked
4. POWER on LED

[Rear Panel - Right Side - QIO-24f, QIO-ML24i, QIO-L24o](javascript:void(0))

(QIO-24f shown)

1. LAN ports (dual) - Up to 1 Gbps per port
2. Exhaust vents (do not block)
3. AC mains inlet connector - Supports universal (international) mains
4. General Purpose Inputs (8 pins, 2-9) 0-24VDC analog, potentiometer, TTL digital or contact closure input.
5. General Purpose Outputs (8 pins, 2-9) Open collector (24V, 0.2A max), with pullups to 3.3V or TTL output
6. Ground Reference (Pin 10 on each connector)
7. 12VDC Source (Pin 1 on each connector). Provides up to 0.2A per pin

[Rear Panel - Left Side - QIO-ML24i](javascript:void(0))

1. Mic/Line Inputs (24 channels - orange) - Balanced or Unbalanced plus +48V (P48) phantom power compliant with IEC 61938
2. Exhaust vents (do not block)

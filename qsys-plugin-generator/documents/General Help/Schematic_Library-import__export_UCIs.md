# Import and Export UCIs

Import and export UCIs to facilitate easy sharing between system designers. You can also download and import UCIs from Q-SYS Library.

[Import UCIs](javascript:void(0))

### Import a UCI

1. Click Tools > Import UCI.
2. Select the .uci file to import, and then click Open.
3. From the User Control Interfaces section of the left-side pane, select the imported UCI.
4. Link or remap the UCI's controls to schematic components, depending on the control type:

   * Toolbox controls â Controls that were added from the Toolbox will appear in the Layers tab with a  icon. Follow the instructions in the "Link Controls" section above to link each control to a component control in your design.
   * Unmapped controls â Controls that were originally added to the UCI from the schematic are indicated with a ? icon. Follow the instructions in [Transferring Settings from one Control to Another](../Q-SYS_Designer/Using_the_Schematic/using_controls.htm#Transfer) to remap the UCI control from a control in the schematic.

[Export UCIs](javascript:void(0))

### Export a UCI

1. From the User Control Interfaces section of the left-side pane, select the UCI you want to export.
2. Click Tools > Export UCI.
3. Select a location on your computer and click Save. The UCI is saved with the .uci file extension.

#### How Does RoomSuite Modular System Work?

RoomSuite Modular System simplifies room deployment by consolidating traditional AV hardware into a single integrated processor that manages audio processing, network connectivity for Q-SYS peripherals, video distribution, and UC platform bridging. Additional devices connect through the RoomSuite modular expander, typically mounted under the table for BYOM laptop connections and peripheral expansion. Setup requires no Q-SYS Designer programming芒聙聰everything is configured through a web browser. Devices are automatically discovered, audio is optimized through built in Room System Optimization, and the system auto generates a user interface based on detected peripherals. Once a room is configured, its settings can be saved as a preset and deployed to other rooms via the cloud using Q-SYS Reflect, enabling centralized remote monitoring and management across all spaces.

#### Q-SYS RoomSuite Manager Software Web Interface With the RoomSuite Modular System

The Q-SYS RoomSuite Manager Software Web Interface is the software provided by the RMP-100, and enables the day-to-day room management, monitoring, and basic configuration of RoomSuite Modular System-enabled rooms and devices.

Capabilities

* View room and device status at-a-glance.
* Perform basic configuration and troubleshooting.
* Manage occupancy, device assignments, and network settings.
* Access logs and simple diagnostics.

Typical Users

IT administrators, facility managers, and support staff who need to monitor and manage rooms without deep system design knowledge.

Common Tasks Performed Through the Interface

* For routine checks on room health and device connectivity.
* To quickly assign or reassign devices to rooms.
* For non-technical adjustments (e.g., occupancy settings, device renaming).
* When remote access is needed for basic troubleshooting.

#### RoomSuite Modular System with an Optional Q-SYS Reflect

Q-SYS Reflect is an optional, cloud based monitoring and management service that can extend a RoomSuite Modular System deployment with large scale visibility and remote administration tools. While RoomSuite runs fully on its own without Reflect, connecting your system to Reflect unlocks additional capabilities designed for multi-room environments.

RoomSuite Modular System does **not** require Q-SYS Reflect to operate. All features芒聙聰including device discovery, room management, optimization, and daily operation芒聙聰is be used entirely through the Q-SYS RoomSuite Manager Web Interface hosted on the RMP-100. Connecting to Reflect is recommended only for organizations that want fleet wide monitoring, preset deployment, or centralized firmware management.

QIO-VEN4

#### Network Microphones (Up to 4 total per room)

* Q-SYS NM-T1 (up to 4 per room); tabletop microphone.

* 3rd party ceiling microphones: Dante Audio Technica ATND1061, Sennheiser TCC M, Sennheiser TCC 2, Shure MXA901, Shure MXA902, Shure MXA920, Yamaha RM-CG (1 per room).

* NL-C4 (4 per room); compact ceiling speaker.

* NL-P4 (4 per room); pendant speakers for open ceilings.

* NL-SB42 (2 per room); soundbar speaker for easy installation under displays.

* NC-90-G2; compact camera for standard meeting rooms.
* NC-12x80; 12x optical zoom and a wide field of view.

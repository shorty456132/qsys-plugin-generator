# Status

The Status page shows key information about the Q-SYS peripheral.

* Device Name: The hostname of the device, as specified in [Network Settings](Network_Settings.htm).
* Design File: The Q-SYS Designer file currently running on the Core with which this peripheral is associated.
* Device Model: The type of Q-SYS peripheral device.
* Firmware Version: The Q-SYS firmware version currently installed on the peripheral.
* Design Uptime: The length of time the peripheral has been connected to the Core running the referenced design file â Days, Hours, Minutes, Seconds.
* Serial Number: Lists the serial number assigned to the device.
* Card Slots: Lists the types of I/O cards installed in Q-SYS peripherals supporting I/O cards.

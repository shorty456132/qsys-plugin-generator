# Import and Export UCI Variables

Import and export UCI Variables to facilitate easy sharing between system designers.

[Import UCI Variables](javascript:void(0))

### Import a UCI Variable

1. Select a UCI from the User Control Interface section.
2. Click Tools > UCI Operations > Import UCI Variables.
3. Imported UCI Variables will show in the selected UCI's Variables tab.

[Export UCI Variables](javascript:void(0))

### Export a UCI Variable

1. Select a UCI from the User Control Interface section.
2. Click Tools > UCI Operations > Export UCI Variables.
3. Select a location on your computer and click Save. The UCI is saved with the .JSON file extension.

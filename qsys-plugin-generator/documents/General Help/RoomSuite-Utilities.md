# Utilities

The Utilities section provides system level tools for monitoring, diagnosing, and maintaining your RoomSuite system, as well as visibility into system behavior, device status, and firmware management.

The Utilities menu includes:

* Event Log - View, filter, and export system events.
* System (Firmware Update) - Download system information, manage Q-SYS authentication options, reboot the processor, and perform firmware updates.

## System (Firmware Update)

The System (Firmware Update) page provides access to system level utilities including system information, authentication options for QSYS Designer integration, processor reboot, and basic firmware update tools.

### General Section

#### System Information Download Button

Downloads system information file.

#### Processor Reboot

Performs a controlled system reboot of the RMP-100, without requiring physical access to the device.

### Firmware Update Section

RoomSuite Modular System supports uploading firmware updates directly through the web interface, which is useful when the System has access to Reflect. Before you begin, obtain the correct firmware from qsys.com and ensure the file version is approved for your deployment.

**Note:** For large or cloud-connected deployments, use Reflect Enterprise Manager for bulk updates. The Utilities page is intended for a single system.

#### Directory Choose Path

Opens file browser to select firmware. After the firmware folder is selected, the RMP-100 prepares the file for upload and installation. Alternatively, you can drag and drop the firmware file folder into the upload area.

During the update, do not power off the RMP-100 processor or disconnect the processor from the network. Avoid closing the browser until the operation is complete.

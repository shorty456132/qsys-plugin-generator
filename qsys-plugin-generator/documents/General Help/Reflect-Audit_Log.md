# Audit Log

The Audit Log provides a list of any user-generated actions initiated either locally on the Q-SYS Core or remotely via Q-SYS Reflect. Only audit events from the Organizations, Domains, and Systems to which you have access as an Organization Owner or Domain Manager are shown.

**Note:** Unlike the Event Log and Alerts table, Audit Log events cannot be cleared.

## What is an Audit Event?

Audit events originate from Q-SYS Reflect.

### Q-SYS Reflect Audit Events

Q-SYS Reflect Audit Events are changes made to the Q-SYS Reflect Organization, Domain, or System by a Q-SYS Reflect user. These include events such as:

* User invitations and invite confirmations.
* Domain creation and deletion.
* Core registration and unregistration.

## Viewing and Filtering Audit Events

You can filter Audit Events by:

* Organization: Restrict the list to show only Audit events from specific Organizations.
* Domain: Restrict the list to show only Audit events from specific Domains.
* System: Restrict the list to show only Audit events from specific Systems.
* User: Restrict the list to show only Audit events triggered by specific users.
* Start date ~ End date: Restrict the list to show only Audit events whose Date & Time stamp is within a target range.
* Search Messages: Restrict the list to show only Audit events with a Message containing a specified text string.

**Tip:** To change the order in which columns appear, click and drag a column name within the Columns menu.

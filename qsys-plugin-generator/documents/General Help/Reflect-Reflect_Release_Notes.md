# Reflect Release Notes

This topic covers release information for Reflect.

[R2025.12](javascript:void(0))

Q-SYS Reflect R2025.12 was launched on December 18, 2025 adding the new features below.

### New Feature Support

[Space Collections](javascript:void(0))

Create custom groups of spaces to monitor system health, simplify navigation, and organize environments based on your operational needs.

* Add Buildings, Outdoor Spaces, Levels, Rooms, and Zones into a single collection
* View system rollups, faults, and status summaries at a glance
* Rename, update, and manage collections directly from Space Manager
* Includes full audit logging for transparency

To learn more, visit [Space Collections](Space_Manager.htm#Space_Collections).

[Drag-and-Drop Space Reorganization](javascript:void(0))

Rearrange your hierarchy instantly using simple drag-and-drop.

* Move spaces to valid parents without opening dialogs
* Auto-scroll, collapsed-parent support, and visual invalid-move rules
* All child spaces, system links, and collections stay intact
* Fully permission-controlled and audit-tracked

To learn more, visit [Space Manager](Space_Manager.htm)

### Updated Features

[System Reliability Enhancements](javascript:void(0))

#### Improved Timeline & Reporting

System Reliability now provides more accurate historical insights, especially when systems are unregistered or reset.

#### Clear Visualization for Unregistered Systems (New!)

Periods when a system is unregistered or deleted now appear as a distinct timeline segment with an info tooltip芒聙聰no more misleading flat lines.

#### Smarter Reliability Reports

* Systems show an info icon when they were unregistered during the selected range
* Tooltips display the exact duration or percentage
* Systems unregistered for the entire period are automatically excluded
* No new state added 芒聙聰 historical data remains clean and uninterrupted

These improvements provide deeper accuracy, reduce confusion, and create a continuous, trustworthy view of system health.

### New Feature Support

[Space Manager](javascript:void(0))

Space Manager is a feature within the Q-SYS Reflect platform that allows users to organize and monitor AV systems based on their physical locations, such as buildings, floors, and rooms. It provides a flexible hierarchy structure for visualizing and managing system deployments geographically. To learn more, visit [Space Manager](Space_Manager.htm).

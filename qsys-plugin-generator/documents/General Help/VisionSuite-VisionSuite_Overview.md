# Q-SYS VisionSuite

From intelligent presenter tracking to intelligent audio-based camera switching, Q-SYS VisionSuite uses multimodal AI to improve the visibility of in-room participants to promote a natural viewing experience for the far-end, empowering teams to feel united and stay engaged no matter where they are.

There are two methods available for deploying VisionSuite: VisionSuite Legacy (Plugins) and VisionSuite Native. Moving forward, VisionSuite Native will be the primary software platform for active development. Transitioning from Legacy to Native will involve reprogramming and commissioning, but it will not incur any additional hardware or software costs. It is important to note that the VSA-100 (which is Q-SYS Native Hardware) can run the plugins or the new Native VisionSuite Designer component. You do not need a SVS1 or SVS4 to run the legacy plugins.

**Note:** At this time, Native does not support divisible rooms or any microphones other than the MXA920 and TCC2. If these features are essential for your needs, you should continue using VisionSuite Legacy until they are implemented in Native.

**Note:** SVS4 and VSA-100 devices cannot be mixed in the same plugin.

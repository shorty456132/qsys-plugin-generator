# Hardware Compatibility: by QDS Version

This page provides an overview of hardware compatibility by Q-SYS Designer Software (QDS) version. It details the timeline of hardware introduction, specifying the version in which each piece of hardware was first supported. Additionally, it outlines when support for each hardware product was discontinued, again indicating the specific version. This information allows for effective tracking of hardware support throughout different QDS versions.

[Understanding Q-SYS Designer Software Version Numbers](javascript:void(0))

Q-SYS Designer Software (QDS) releases follow a three-level numbering convention â *major release*.*minor release*.*maintenance release* (for example, 9.4.2). Additionally, you can view the specific build number of the release by clicking Help > About. Note that:

* If you open a design file saved in a newer version of QDS than the one you are currently running (for example, a 9.5.0 design in version 9.4.2), you are warned that certain functionality within the design might not function. This is not recommended. Instead, install the version of QDS that matches the version of the design.
* If you open an older design file in a newer version of QDS (for example, a 9.4.2 design in QDS 9.5.0) and then attempt to save it, you are warned that doing so might make the design not load correctly in the previous version.

**CAUTION:** Make sure your design files are backed up using your current release before starting the upgrade to a newer release.

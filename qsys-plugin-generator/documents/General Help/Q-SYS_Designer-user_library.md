# User Components

The User Components list allows you to configure a component from the Schematic Elements list, then place it in the User Components list for use in the current or other designs - it is not design specific.

[Add a Component to the User Components list](javascript:void(0))

1. Place a component in the schematic area and configure its Properties as desired.
2. Select and drag the component into the User Components list, or select the component and click the + (plus sign) in the User Components heading. The User Component dialog displays.
3. In the User Component dialog, enter a name for the component. The default is "User Component 1". The number will sequence unless you give it a name without a number. The components are sorted alphabetically, so you can name or rename the user component for sorting purposes
4. Press OK to add the component, or Cancel to exit without adding the component.

You can configure the same kind of component in a different way, and add it to the list giving you two variations of the same component.

[Add a Folder to the User Components list](javascript:void(0))

Click the folder icon  in the User Components heading, a folder named "New Folder" is created.

You can add components to the folder by dragging and dropping a component onto the new folder, or select the folder and use the Add a Component procedure above.

[Using the User Components](javascript:void(0))

Simply select the component you want, and drag it into the Schematic as you would from the Schematic Elements. The component is not removed from the User Components list.

[Delete a Component or Folder from the User Components list](javascript:void(0))

Select the component or folder you want to delete, and press the Delete Key, or click the - (minus sign) in the User Component heading.

[Install a User Component on Another PC](javascript:void(0))

When you create a User Component, it is stored on your computer in the Documents folder for your user name. You can easily copy the User Component file and install it in another location.

1. Create a new User Component.
2. In Windows Explorer, go to `C:\Users\user-name\Documents\QSC\Q-Sys Designer\User Components`.
3. Copy (or move) the .quc file for your User Component to a new location.
4. Double-click the .quc file to install it in Designer.

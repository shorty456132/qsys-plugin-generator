# Asset Installer

Asset Installer is the place to manage, install, and update your Checked Out assets from the Q-SYS Library.

**Note:** Q-SYS Designer Software v10.1 or higher is required to use Asset Installer.

**Note:** For best results and to avoid version conflicts, please remove older versions of plugins that were installed via Asset Manager/previous versions of Q-SYS Designer.

## Accessing Asset Installer

1. Click Sign In on the upper right corner of Q-SYS Designer.
2. Using your QSC ID, sign in.

   **Note:** Once signed in, you will have access to your checked out assets in Q-SYS Library.
3. In Q-SYS Designer, click the **Tools** menu.
4. Select **Show Asset Installer**.

Read below to learn about the information and controls available in Asset Installer.

[Available Assets](javascript:void(0))

This tab shows assets that have been Checked Out in Q-SYS Library and are available to install/update with the following properties.

[Toolbar](javascript:void(0))

#### Search assets

Enter text here to find the desired asset.

#### Install All

Click to install all listed assets.

#### Install Selected

Click to install assers with the select checkboxes checked.

#### Refresh

Click to refresh the list for status changes.

[Window Properties](javascript:void(0))

#### Asset

Displays the manufacturer of the asset.

#### Description

Displays the name and information about the asset.

#### Local Version

Displays the asset version currently installed on the local PC.

**Note:** Information is only displayed if a version of the asset is currently installed on the PC.

#### Latest Version

Displays the latest available version of the assets in Q-SYS Library.

#### Install Status

Displays if the latest version of the asset is installed

#### Select / Select All Checkbox

* The top box selects all listed assets.
* Use the individual boxes to select the desired assets.

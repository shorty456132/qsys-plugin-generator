# Script Programmer Mode

Script Programmer Mode, accessible from the Tools menu, is used in conjunction with the Code Name / Script Access properties. It allows you to quickly view the Script Access setting directly on the component in the design schematic without the need to disconnect from the Q-SYS Core processor.

## Script Access Setting

Based on the setting for the [Code Name and Script Access](../Control_Scripting/Code_Name_Script_Access.htm#Code) property, components will display as the following when Script Programmer Mode is enabled.

[All](javascript:void(0))

[External](javascript:void(0))

[None](javascript:void(0))

[Script](javascript:void(0))

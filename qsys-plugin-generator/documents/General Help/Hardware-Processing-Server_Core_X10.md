# Server Core X10

The Server Core X10 is a high-performance, compact Q-SYS processor designed to power medium-sized projects with ease. It delivers the full capabilities of Q-SYS processing without the need for scaling licenses. Housed in a sleek 1RU rackmount chassis with a front-panel display, multiple network ports, and USB connectivity, Server Core X10 offers seamless integration into any Q-SYS system.

[Features](javascript:void(0))

* 256 x 256 Network Audio Channels
* Up to 128 x 128 Software Dante audio channels (8 x 8 included)
* 64 AEC Processors
* Up to 128 Multitrack audio players (16 channels by default)
* 32 VoIP instances

### Front Panel

1. **Power light** - lights up blue when the unit is powered on
2. **Front-panel display** - displays pertinent information about the core, its network configuration, the system it is running, active faults, etc.
3. **Navigation buttons (up, down, left, right)** - these buttons allow you to navigate through the menus on the front-panel display:
   1. The up and right buttons do the same thing: advance you to the next menu item
   2. The down and left buttons do the same thing: go back to the previous menu item
4. **ID/Select button** - the center button is analogous to the ID button on other Q-SYS Core processors. Press to put the Core into ID mode for identification within Q-SYS Designer Software. Press again to turn off ID mode.

### Rear Panel

1. **HDMI port**: not supported.
2. **USB A and USB C ports**: not supported.
3. **Serial communications RS232 (male DB-9)**: for connecting to serial devices.
4. **Q-SYS LAN ports (RJ45)**: from left to right; top row is LAN A and LAN B, bottom row is LAN C and LAN D.
5. **Power supply unit (PSU)**.

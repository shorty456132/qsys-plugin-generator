# Accessing Q-SYS Administrator

[Administrator](AdministrationInterface.htm) is accessible within Q-SYS Designer, or separately as a stand-alone application.

**Note:** If Access Control is enabled in Q-SYS Core Manager, users must log in to gain access to all Administrator functions. See [Administrator Access Control](#Administ).

[Q-SYS Designer](javascript:void(0))

In Q-SYS Designer, you can access Q-SYS Administrator in Run or Emulate mode.

Click **Tools** > **Show Q-SYS Administrator**.

[Stand-alone Application](javascript:void(0))

The Q-SYS Administrator stand-alone application is included with the Q-SYS Designer download. You can access designs in Run mode only. Q-SYS Designer is not required when you use the stand-alone application, allowing you to perform administration tasks without allowing access to the design running on the Q-SYS Core.

In the stand-alone application, click **File** > **Administer Design**. (Design names with an asterisk indicate unsaved changes.)

[Administrator Access Control](javascript:void(0))

Access to Q-SYS Administrator is governed by Q-SYS Core Manager users and user roles:

* If Access Control is enabled, any user with the role of 'Administrator' or 'Technician' can use Q-SYS Administrator and all of its functions.
* If Access Control is disabled, anyone can access Q-SYS Administrator and all of its functions.

To learn how to control access in Q-SYS Core Manager, see the Users topic in the Q-SYS Core Manager help.

[Switching Users and Logging Off](javascript:void(0))

While logged in to Q-SYS Administrator, you can switch to another Q-SYS Core user or log off Administrator:

1. In the lower-right corner of the Administrator window, click your user name.
2. Select **Log Off** or **Switch User**.

**Note:** If Access Control is disabled in Q-SYS Core Manager, the Guest user is logged in by default. You cannot log off or switch from this account.

# Zoom Compatibility

> Source: https://help.qsys.com/MicroContent/Resources/MicroContent/qsys-micro-content/zoom-compatibility.htm

#### Zoom Compatibility

If you are using the [HID Conferencing](../../../../Content/Schematic_Library/usb_telephony.htm) component to control a remote computer running Zoom or Google Meet, set this property to 'Enabled.' Otherwise, when controlling another conferencing app such as Skype for Business, set to 'Disabled' (default). When this property is enabled, the USB Bridge Name property is disabled.

**Tip:** Enabling Zoom Compatibility may improve the user experience with macOS conferencing solutions.

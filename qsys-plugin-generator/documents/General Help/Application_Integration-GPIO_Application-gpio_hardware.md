# GPIO Q-SYS Supported Hardware

The following table lists Q SYS hardware that supports General Purpose Input/Output (GPIO) functionality, including the number of ports, connector types, and compatibility details for cores, peripherals, amplifiers, and control devices.

| Hardware | GPIO | GPIO In | GPIO Out | Number of Ports | Pin Connector |
| --- | --- | --- | --- | --- | --- |
| **Cores** | | | | | |
| Core 24f |  |  |  | 8 GPIO inputs and outputs | RS-232: Three-pin Euro terminal connection |
| NV-32-H (Core Mode) |  |  |  | 2 inputs and 3 outputs | RS-232: Three-pin Euro terminal connection |
| 8 Flex |  |  |  | 8 GPIO inputs and outputs | RS-232: Three-pin Euro terminal connection x2 |
| 110f [1](#1.) |  |  |  | 16 GPIO inputs and outputs | RS-232: Three-pin Euro terminal connection |
| 510i |  |  |  | 2 Female 15-pin D shell connectors | RS-232: Male 9-pin D shell connector |
| **Peripherals** | | | | | |
| *Audio - Q-LAN* | | | | | |
| I/O-Core 24f |  |  |  | 8 GPIO inputs and outputs | RS-232: Three-pin Euro terminal connection |
| QIO-FLEX4A |  |  |  | 4 inputs and 8 outputs | RS-232: Three-pin Euro terminal connection |
| DCIO / DCIO-H |  |  |  | 6 GPI inputs and 4 relay outputs | RS-232: Three-pin Euro terminal connection |
| *Amplifiers - Q-LAN* | | | | | |
| CX-Q, CXD-Q, DPA-Q |  |  |  | 8 configurable | 16-pin Euro style connector |
| SPA-Qf |  |  |  | 4 configurable | 6-pin Euro receptacle |
| *Video* | | | | | |
| NV-32-H (Peripheral Mode) |  |  |  | 2 inputs and 3 outputs | RS-232: Three-pin Euro terminal connection |
| *Control* | | | | | |
| QIO-GP8x8 |  |  |  | 8 inputs and outputs | 10-position black Euro connector |
| QIO-LVR4 |  |  |  | 4 relay circuits | 12-pin, 3.5 mm Euro style connector with three pins per relay circuit: Normally Open (NO), Common (C), and Normally Closed (NC) |

###### 1. The Core110f V1 Processor allows for GPIO. The V2 does not.

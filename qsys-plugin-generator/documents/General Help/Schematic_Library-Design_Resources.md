# Design Resources

Use the Q-SYS Designer Software > Tools > Design Resources menu to manage UCI Styles and External Lua Modules for a design.

#### [UCI Styles](uci_styles.htm)

Install custom CSS files to a design, and then apply a selected style to a user control interface (UCI).

#### [External Lua Modules](../Control_Scripting/External_Lua_Modules.htm)

Install Lua modules to a design for use by scripts or plugins.

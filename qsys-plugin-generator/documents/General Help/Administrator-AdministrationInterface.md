# Administrator

Q-SYS Administrator allows you to configure various Q-SYS resources, including the Public Address (PA) System, Commands, Command Scheduling, and Users (for Page Stations, External Control Protocol, and File Management Protocol). To learn about the multiple ways you can access Administrator, see [Accessing Q-SYS Administrator](Accessing_Administrator.htm).

#### [PA Global Settings](PA_Global_Settings.htm)

Configure settings that affect several areas of the PA system, including the Priority Mode and Priority Levels for Page Stations, Users, and Commands.

To learn about the PA system, see [Q-SYS PA System Overview](PA_System_Overview.htm).

#### [Page Stations](Page_Stations.htm)

Configure settings for Page Stations in the current Q-SYS design. The PS-TSCG3 identifies as a Virtual Page Station on Input *n* within Administrator.

#### [PA Zones](PA_Zones.htm)

This tab displays each of the paging zones you have established as audio outputs on the PA Router in the current Q-SYS design. Use [Tags](Tags.htm) to group the zones, making it easier to assign zones to Commands.

#### [Commands](Commands.htm)

Create commands that you assign to page stations, including Page, Play Messages, Control Change, and Load Snapshot commands. Assign [Tags](Tags.htm) to commands to assist in assignment to Users or Page Stations.

#### [Command Schedule](Command_Schedule.htm)

Schedule commands by creating an events containing the command start time/date, repeat period, recurrence, and the end time/date.

#### [Users](Users.htm)

Create users in Q-SYS Administrator to control access for External Control and File Management. In addition, Administrator users can be assigned PA system paging capabilities, including the ability to override Page Station settings and invoke selected commands.
